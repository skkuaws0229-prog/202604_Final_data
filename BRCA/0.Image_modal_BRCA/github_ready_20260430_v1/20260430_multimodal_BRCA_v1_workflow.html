🔬 이미지 전처리 상세

  • TCGA GDC Portal에서 BRCA H&E WSI 다운로드 (~1,000장)
  • CLAM으로 조직 영역 자동 검출 + 배경 제거
  • 256×256 타일링 (20x magnification)
  • UNI2 (ViT-Large, frozen backbone)로 타일당 1,024d 임베딩
  • Mean pooling (baseline) 또는 ABMIL (고도화) → 슬라이드당 1,024d
  • 결과를 .h5 파일로 저장, patient_id로 매핑

📐 입력 텐서 구조

  • x_omic = RNA(978d) + Pathway(200d) = 1,178d
  • x_drug = Morgan(1024d) + Target(300d) + LINCS(978d) + KG(100d) = 2,402d
  • x_image = Slide embedding(1024d) — GDSC에서는 zero vector
  • Stage 1 입력: concat(x_omic, x_drug) = 3,580d
  • Stage 2 입력: pIC50(1d) + x_image(1024d) + RNA_summary(200d) = 1,225d

🧠 Missing Modality 처리

  • GDSC 셀라인: WSI 없음 → Stage 1만 사용 (이미지 불필요)
  • TCGA 환자: Stage 1 예측값 + 이미지로 Stage 2 보정
  • 이미지 없는 환자: Stage 2 스킵, Stage 1 score 그대로 사용
  • 향후 modality dropout 적용 시 통합 학습 가능

✅ Ablation 실험 계획

  • Baseline: omic + drug (기존, 이미지 없음)
  • +Image (mean): Stage 2에 mean pooling embedding 추가
  • +Image (ABMIL): Stage 2에 ABMIL embedding 추가
  • Image only: 이미지만으로 약물 감수성 예측
  • 평가: METABRIC / PRISM 외부 검증 + Spearman ρ 비교
